# Analysis

This directory contains Jupyter notebooks to reproduce all results and figures from the paper.

## Contents

- [mhn.ipynb](mhn.ipynb) contains the plots of the parameter matrix for the cMHN and oMHN (fig. 1).
- [predictions.ipynb](predictions.ipynb) contains MHN predictions (fig.s 2 and 3)
- [metmhn.ipynb](metmhn.ipynb) contains the plot of the parameter matrix for the metMHN (fig. 4).
- [posterior.ipynb](posterior.ipynb) contains the posterior analysis of an MHN (fig. 5).
- [plotting.py](plotting.py) contains plotting utilities and config.
- [results](results) contains the plots.
